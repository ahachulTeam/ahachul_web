# Ah Hachul Frontend Repo

## 개발 스택

- Typescript
- Next.js 
- React-query, recoil, axios
- styled-components

## 패키지 구조

```javascript
|-- ahachul_frontend
```

## 브랜치 전략

```javascript
|-- main
    |-- develop
        |-- <label>/<#issue number>/<한글>
    |-- hotfix
```

## 협업 규칙

### 커밋 메시지

- [gitmoji 공식문서](https://gitmoji.dev/)
- **`<label>: <commit message> (#issue number)`**

### 이슈

1. 이슈 생성 후 PR
2. 코드 리뷰를 통한 피드백 후 `approve`
3. `develop` 브랜치 `merge`
