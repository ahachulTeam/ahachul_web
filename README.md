## 시작하기

```
# 로컬 환경으로 클론
git clone https://github.com/ahachulTeam/ahachul_web.git

# 디펜던시 설치
yarn

# 개발 서버 실행
yarn ahhachul
```

개발 서버가 실행되면 브라우저에서 [http://localhost:3000](http://localhost:3000/) 로 접속하실 수 있어요.
