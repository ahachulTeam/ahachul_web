# Toast

## Installation

`@ahhachul/react-components-toast` 패키지를 설치합니다.

<Source
  dark
  language="bash"
  code={`
    yarn add @ahhachul/react-components-toast
  `}
/>

## Usage

1. Toast를 노출할 영역에 `ToastProvider`를 추가합니다.

<Source
  dark
  language="jsx"
  code={`
<ToastProvider>
  <Router /> // e.g. React Router Dom
</ToastProvider>
  `}
/>

2. `@ahhachul/react-components-toast/style.css`를 불러옵니다.

<Source
  dark
  language="jsx"
  code={`
import "@ahhachul/react-components-toast/style.css";
  `}
/>

3. `useToast`를 사용하여 `Toast`를 노출합니다.

<Source
  dark
  language="jsx"
  code={`
const SomeComponent = () => {
  const { toast } = useToast();
  
  const handleClick = () => {
    toast({
      payload: {
        message: "toast",
      },
    });
  };

return (<Button onClick={handleClick}>토스트 버튼</Button>);
};
`} />
